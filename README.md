# Парсинг политических партий

## Описание
Этот проект парсит страницу с политическими партиями на сайте Минюста России и извлекает информацию о партиях и их документах (уставы, регистрации и т.д.).

## Как запустить проект

1. Установите зависимости:
       pip install -r requirements.txt
    
2. Запустите скрипт:
       python parse_parties.py
    
3. Результаты будут сохранены в файл parties.json.
